## Project Description
An AI-powered web application that converts voice notes into text and generates concise summaries with key points and action items.

## Features
- Upload voice notes (mp3, wav)  
- Speech-to-text transcription using AssemblyAI  
- AI-generated summaries using OpenAI GPT  
- Clean, interactive Streamlit interface  

## Tech Stack
- Python  
- Streamlit  
- AssemblyAI API  
- OpenAI GPT  
- NLP and Generative AI  

## How It Works
1. Upload an audio file  
2. Audio is transcribed into text  
3. AI generates a concise summary  
4. Key points are displayed to the user
