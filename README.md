# AI Voice Note Summarizer

An AI-powered web application that converts voice notes into text and generates concise summaries with key points and action items.

## Features
- Upload voice notes (single-speaker, clean audio)
- Automatic speech-to-text transcription using AssemblyAI
- Intelligent summarization using OpenAI GPT
- Simple and clean Streamlit web interface

## Tech Stack
- Python
- Streamlit
- AssemblyAI (Speech-to-Text)
- OpenAI GPT (Text Summarization)

## How It Works
1. User uploads an audio file
2. Audio is transcribed into text
3. AI model summarizes the transcription
4. Key points and action items are displayed

## Use Case
Designed for productivity use cases such as meeting notes, voice reminders, and personal task tracking.
